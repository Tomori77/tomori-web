# 设计风格指南 (DESIGN.md)

## 1. 整体理念
- **主题**：浅色、明亮、空气感
- **风格**：柔和渐变 + 毛玻璃质感 + 液态玻璃流动感 + 轻量几何装饰
- **体验**：高级、干净、有呼吸感，交互细腻但不干扰阅读，响应迅速，微动效带来愉悦感
- **多端一致性**：桌面、平板、手机体验统一，布局灵活适配

## 2. 色彩系统

### 2.1 主色调
| 角色 | 颜色值 | 说明 |
|------|--------|------|
| 背景基色 | `#F8FAFC` | 全局浅灰白，柔和 |
| 卡片/毛玻璃背景 | `rgba(255,255,255,0.6)` ~ `0.8` | 半透白，配合模糊 |
| 主渐变 | `linear-gradient(135deg, #667eea 0%, #764ba2 100%)` | 蓝紫渐变，用于主要按钮、强调元素 |
| 辅助渐变 | `linear-gradient(135deg, #f093fb 0%, #f5576c 100%)` | 粉红暖渐变，用于次要强调、标签 |
| 清新渐变 | `linear-gradient(135deg, #4facfe 0%, #00f2fe 100%)` | 青蓝渐变，用于信息提示、链接 |
| 文本主色 | `#1E293B` | 深蓝灰，高可读性 |
| 文本次要色 | `#64748B` | 中性灰 |
| 边框色 | `rgba(255,255,255,0.3)` | 半透白边框，用于毛玻璃边缘 |
| 阴影 | `0 8px 32px rgba(0,0,0,0.05)` | 轻阴影增加悬浮感 |

### 2.2 液态玻璃专属色
- 装饰性透明光斑可使用多彩渐变，带透明度和模糊：
  - `radial-gradient(circle at 30% 30%, rgba(102,126,234,0.15), transparent 50%)`
  - 作为背景装饰层，营造流动液体感。

## 3. 字体与排版
- **首选字体**：`'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif` （现代、清晰）
- **标题字重**：600 ~ 700
- **正文字重**：400，行高 1.6 ~ 1.8
- **代码/编辑器字体**：`'JetBrains Mono', 'Fira Code', monospace`

## 4. 多端适配 (Responsive Design)

### 4.1 断点定义
采用移动优先策略，主要断点：
- **手机**：< 640px (sm)
- **平板**：640px ~ 1024px (md)
- **桌面**：> 1024px (lg)
- **大屏**：> 1280px (xl) 用于超大显示器

### 4.2 布局变化
- **手机**：单列布局，卡片、表单宽度100%，间距缩小为16px，导航转为底部固定或顶部汉堡菜单。
- **平板**：两列网格（如文章列表），侧边栏可能收起为抽屉。
- **桌面**：多列网格（3列或4列卡片），侧边栏固定显示，最大内容宽度1200px居中。
- **管理后台**：大屏下表格全宽显示；手机端表格变为卡片式堆叠，每行数据以键值对展示。

### 4.3 导航适配
- 桌面：顶部导航栏水平排列，毛玻璃效果，可容纳多个链接。
- 平板/手机：使用汉堡菜单（☰），点击后弹出毛玻璃全屏抽屉，菜单项纵向排列，动画为淡入 + 轻微位移。
- 底部导航栏（可选）：手机端可显示3~5个核心操作（首页、写文章、我的、管理），用图标+文字，高度50px左右，同样毛玻璃风格。

### 4.4 编辑器适配
- 编辑器在手机端应占用全部视口高度，工具栏简化或折叠，预览区域可全屏切换，保证输入体验。
- 通过 `touch-action: manipulation` 防止双击缩放，优化移动端点击响应。

### 4.5 交互与性能
- 所有动画和过渡在移动端保持60fps，避免使用 `backdrop-filter` 过多导致卡顿，必要时降级为半透明纯色背景。
- 使用 `will-change` 提前通知浏览器，但谨慎使用。
- 图片和资源按需加载，移动端提供缩略图，点击放大。
- 表单元素字号不小于16px，防止iOS缩放。

### 4.6 测试标准
- 在320px ~ 1920px宽度内无横向滚动条，所有内容可读。
- 触摸区域不小于44x44px（符合WCAG）。
- 使用 `prefers-reduced-motion` 媒体查询关闭或减弱动画，尊重用户偏好。

## 5. 毛玻璃组件风格
以下为通用组件样式，均采用 `backdrop-filter: blur()` + 半透明背景 + 细边框。

### 5.1 导航栏
- 固定顶部，背景 `rgba(255,255,255,0.7)`，`backdrop-filter: blur(20px)`，下阴影 `0 1px 4px rgba(0,0,0,0.02)`
- 链接：默认浅灰文字，悬停时渐变文字（主渐变）或底部渐变色下划线动画

### 5.2 卡片（文章列表、资料卡）
- 背景 `rgba(255,255,255,0.6)`，`backdrop-filter: blur(12px)`
- 圆角 `16px`，边框 `1px solid rgba(255,255,255,0.4)`
- 悬浮时：`transform: translateY(-2px)`，阴影加深，可能伴随轻微亮度变化
- 内部留白 `24px`（移动端缩减为 `16px`）

### 5.3 按钮
- **主按钮**：主渐变背景，白色文字，圆角 `12px`，无边框，悬浮时缩放 `scale(1.02)`，阴影增强。
- **毛玻璃按钮**：背景 `rgba(255,255,255,0.5)`，`backdrop-filter: blur(10px)`，边框半透白，渐变文字或图标，悬浮后背景不透明度增加。
- **液态玻璃按钮**：极圆润（border-radius: 20px+），背景半透明并叠加一层微光动画（如移动的径向渐变），点击时有水波纹扩散效果。

### 5.4 输入框/编辑器
- 背景 `rgba(255,255,255,0.7)`，`backdrop-filter: blur(8px)`，圆角 `12px`，边框 `1px solid rgba(255,255,255,0.5)`，聚焦时边框渐变色。

## 6. 动画与微交互

### 6.1 跳动效果
为关键元素添加活泼但不烦人的跳动动画：
- **心跳跳动**：用于“喜欢”、“新消息”图标或数字。
  ```css
  @keyframes heartbeat {
    0%, 100% { transform: scale(1); }
    25% { transform: scale(1.2); }
    50% { transform: scale(1); }
    75% { transform: scale(1.1); }
  }
  .heartbeat { animation: heartbeat 1.5s ease infinite; }
  ```
- **轻微浮动**：卡片或装饰方块悬浮时上下浮动 `3~5px`，循环时间 3~4s。
  ```css
  @keyframes float {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-6px); }
  }
  .float { animation: float 3s ease-in-out infinite; }
  ```

### 6.2 渐变流动
为液态玻璃或背景添加缓慢移动的渐变：
- 背景 `background-size: 200% 200%`，配合动画移动位置。
  ```css
  @keyframes gradientShift {
    0% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
    100% { background-position: 0% 50%; }
  }
  .shifting-gradient { animation: gradientShift 8s ease infinite; }
  ```

### 6.3 页面过渡
- 前端路由切换时，容器淡入淡出（opacity 过渡 0.2s）或轻微上滑进入（translateY 10px → 0），切忌大范围移动。

### 6.4 小方片装饰
- 在背景或卡片角落放置小的彩色矩形块（8~20px），旋转一定角度，使用渐变或纯色半透明，可应用 `float` 动画或缓慢旋转。
- 多个小方片不同延迟、不同大小，营造灵动科技感。

## 7. 布局与间距
- 最大内容宽度 `1200px`，居中。
- 间距以 `8px` 为基准：小间距 `8px/12px`，标准 `16px/24px`，大留白 `32px/48px`。
- 使用网格系统：卡片列表采用 CSS Grid，自适应列数。
  ```css
  .grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)); gap: 24px; }
  ```

## 8. 图标与装饰
- 推荐使用轻量级 SVG 图标（如 Feather Icons 或 Phosphor Icons），线条简洁，色彩可跟随文字渐变。
- 装饰性几何图形：透明圆环、弧形、点状背景，使用绝对定位 `z-index: -1`，避免影响交互。

## 9. 响应式与无障碍
- 保持所有毛玻璃效果在移动端性能良好（测试模糊是否卡顿），必要时降级为纯色半透明。
- 对比度满足 WCAG AA 标准（尤其文本与毛玻璃背景之间），适当加深文字颜色或加半透明底色。
- 所有交互元素需有 `:focus-visible` 样式，键盘可导航。

## 10. 实现建议
- 使用 CSS 自定义属性（变量）管理颜色和模糊数值，方便超级管理员在后台动态切换（虽然不强制）。
- 动画使用 `prefers-reduced-motion` 媒体查询，为偏好减少动效的用户关闭或减弱。
- 多端适配的核心在于使用相对单位（`rem`, `%`, `vw`）、弹性盒和网格布局，避免固定宽度。
